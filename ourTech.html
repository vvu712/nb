NUTRIWISE BOT WORKFLOW

Leveraging OpenLLama to Generate Responses

Processing User Input

NutriWise Bot leverages OpenLLaMA, an advanced large language model,for its sophisticated natural language processing techniques like tokenization and self-attention mechanisms to analyzes user input with exceptional accuracy. Trained on over 1.2 trillion tokens, including RedPajama pre-trained data sets, this model outperforms previous iterations, enabling NutriWise Bot to provide precise and personalized nutrition guidance to users.

Refining NutriWise Bot's responses

By training the OpenLLaMA language model with specialized datasets focused on nutrition dialogue and cultural insights that is overviewed by nutrition experts, the bot ensures accuracy and relevance in its recommendations. Integration with the Nutritionix API further enriches responses with detailed nutrition information for common foods, enhancing user experience and trust.

NutriWise Bot Image Recognition Technique


NutriWise Bot utilizes Hybrid Task Cascade (HTC), a deep learning technique, the API accurately identifies objects and shapes within images. By leveraging a food recognition API that is continuously trained on user-submitted photos from MyFoodRep, the bot can aid in food logging and dietary analysis. NutriWise Bot ensures accuracy and accessibility, with a growing repository that considers diverse cultural food preferences and choices.
